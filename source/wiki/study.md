(A) Write the output of the list when the following pseudocode is executed for the list (list = [0,1,2,3,4,5,6,7,8]).

```
------- [疑似コード（Pseudocode）]-------

list = [0,1,2,3,4,5,6,7,8]

function(list):
    THRESHOLD = 3
    temp_val = 0
    i = 0
    results = []

    for val in list:
        i = i + 1
        temp_val = temp_val + val

        if i >= THRESHOLD:
            results.append(temp_val / THRESHOLD)
            i = 0
            temp_val = 0

    return results
------------------------------------------------
```
